---
title: 'Semicolon in Install Path'
description: 'MTA detected a semicolon character in the installation path. We advise you to move the MTA or GTA installation in this case.'
tableOfContents: false
---

MTA detected a [semicolon character](https://en.wikipedia.org/wiki/Semicolon) `;` in the installation path of either MTA or GTA.
If you experience problems when running MTA, move your installation to a path that does not contain a semicolon.
